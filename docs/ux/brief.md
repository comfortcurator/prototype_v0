# UX Brief

## Experience Principles
- **Premium calm:** Emulate Airbnb’s aesthetic with stoic, refined visuals and purposeful whitespace.
- **Actionable clarity:** Surface health status, alerts, and next steps prominently.
- **Trust first:** Reinforce secure transactions, compliance, and automation reliability with microcopy.
- **Local relevance:** Highlight INR pricing, GST inclusion, and India-centric logistics.

## Primary Flows
1. **Host Onboarding**
   - Landing hero → Sign up → Property import (Airbnb URL/manual) → Verification → Dashboard tour.
2. **Property Health Monitoring**
   - Dashboard cards → Map selection → Health timeline → Suggested actions (cleaning, reorder).
3. **Marketplace Purchase**
   - Browse bundles/items → Add to cart → Configure package → Razorpay checkout → Order confirmation.
4. **Subscription Management**
   - Tier comparison → Property assignment → Payment setup → Renewal reminders.
5. **Admin Catalogue Management**
   - Item creation → Inventory session logging → Package builder → Analytics review.

## Wireframe Notes
- **Dashboard:** Split layout with summary KPIs, map, property carousel, alert feed.
- **Marketplace:** Grid cards with hover micro-interactions, sticky order summary sidebar.
- **Checkout:** Multi-step with progress indicator, address confirmation, Razorpay widget, GST invoice preview.
- **Admin overview:** KPI tiles, revenue chart, table of critical alerts, quick links to sessions and orders.

## Validation Plan
- Conduct usability tests with 3–5 Indian hosts (tier-1 & tier-2 cities) using clickable prototype.
- Run accessibility review (screen reader + keyboard) before development handoff.
- Collect NPS and SUS scores during closed beta; iterate before public launch.
